# Gossips !

## :wrench: Installation guide
You can install the useless-but-awesome student project by doing :

    $ git clone https://github.com/Crysicia/THP-Gossips.git
    $ cd THP-Gossips
    $ bundle install

Gossips is a simple database-only story-posting app.
